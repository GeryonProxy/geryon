Geryon

High-performance, multi-database connection pooler and proxy for PostgreSQL, MySQL, and MSSQL. Built in pure Go with zero external dependencies.

⚡ Pure Go 🚀 Zero Dependencies 📊 Web Dashboard 🔒 TLS/mTLS
Get Started View on GitHub

Three Bodies, One Mind

Named after the three-bodied giant of Greek mythology, Geryon speaks PostgreSQL, MySQL, and MSSQL wire protocols from a single static binary.

⚡

High Performance

Sub-millisecond latency overhead. Connection pooling, query caching, and prepared statement optimization.

🔌

Three Protocols

Native support for PostgreSQL, MySQL, and MSSQL wire protocols. No drivers needed.

📊

Web Dashboard

Real-time monitoring with built-in web dashboard. View connections, queries, cache stats, and cluster health.

🔒

Enterprise Security

TLS termination, mTLS client authentication, SCRAM-SHA-256, and per-user connection limits.

☸️

Raft Clustering

Built-in Raft consensus for configuration replication and automatic failover across multiple nodes.

🤖

MCP Server

Native MCP (Model Context Protocol) server for Claude Desktop and Claude Code integration.

Quick Start

Install Geryon with a single command and start proxying in minutes.

Install

# Install with one command
$ curl -sSL https://raw.githubusercontent.com/GeryonProxy/geryon/main/install.sh | bash

# Or with Homebrew
$ brew tap geryonproxy/geryon
$ brew install geryon

# Or with Docker
$ docker pull geryonproxy/geryon:latest
